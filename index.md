# Homeshell

Welcome to Homeshell!

## Features

Homeshel includes

- **Officeworks** (a word processor to create documents)
- **WebBox** (a simple web browser)
- **tonline** (social networking)
- **3rd and 1st Party App Support** (for safer and faster apps)
